Tests

The tests below are designed to work within a running local server. Please use SimpleHTTPServer from the root directory of onfontready, and navigate to the page you are currently viewing relative to the local server (example: http://localhost:8000/tests/). Please refer to the document TestingOldIE.md for instructions about running these tests on legacy versions of Internet Explorer.

Usage Tests

  • Main Tests - Exhaustively tests features and failure cases of both the modern and legacy versions of onfontready.

Development Tests

  • New Browser Memory Leak Test - Tests memory usage of event listeners after shutdown.
  • contentWindow Availability Test - At what point is an iframe's contentWindow available?
  • Space Character Widths - How wide is the space character in various browsers and generic fonts?
  • Inline Absolute Overflow Test - What browsers are capable of making an absolutely positioned element fit inside a relatively positioned inline element?
  • Scroll-Based Resize Detection Test - Which browsers can detect font-size triggered scroll events?

Old Tests

  • Browser Memory Leak Test (deprecated) - Tests memory usage of event listeners, but the results aren't relevant to onfontready's usage.
